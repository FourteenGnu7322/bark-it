What the User can do

The User is allowed to make posts with images, modify their profile picture, and their profile with something that follows the guidelines.

What the User can't do

The User isn't allowed to exploit the api, break the guidelines, or scam individuals.

Content that contains gore, nudity or a sexual meaning

Any content containing gore, must be marked with the 18+ tag, if such content can't be posted under such tag, (for example, profile pictures) then it doesn't apply and will result in a warning. Some cases may apply even in a text format.

Some examples of gore, nudity or content with a sexual meaning are

  • Content with visible human organs.
  • Content displaying the separation of a human and a body part.
  • Content displaying humans suffering physically.
  • Content displaying one's genitals.
  • Content displaying sexual sounds.

Bannable content that contains gore, nudity or a sexual meaning (even with 18+ tag, simply because they are illegal)

  • Zoophilia - content that displays anything showing off an animal or an animal like being in a sexual manner
  • Necrophilia - content showing off a corpse of a human in a sexual way
  • Pedophilia - content exploiting children in a sexual way

Exploiting the api

The api exists for people to make bots and clients for personal use. If it is used to crash or cause a heavy load on the server, it will lead to an ip ban from all of Bark-it's services.

Some examples of Exploiting the api

  • Sending a mass amount of packets containing post data
  • Uploading images intended to crash other users
  • Crashing the service
